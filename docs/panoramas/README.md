# 全景图占位说明

请把 2:1 比例的等距柱状全景图放在本目录，并使用以下文件名：

- `village_gate.jpg`
- `green_path.jpg`
- `classroom.jpg`
- `farm_product.jpg`

推荐尺寸：`4096x2048`、`6000x3000` 或其他 2:1 比例图片。

如果没有这些图片，页面会显示“请将 2:1 比例全景图放入 public/panoramas 文件夹”，不会崩溃。
